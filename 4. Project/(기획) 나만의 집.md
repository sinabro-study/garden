### 일반 사용자

로그인 페이지 -> 메인 페이지 흐름

1. 사용자는 회원가입할 수 있다.
	1. 회원가입 시 1,000 포인트를 지급받는다.
2. 사용자는 로그인할 수 있다.
	1. 로그인할 때마다 1일 1회 50 포인트를 지급받는다.

**고려사항**
- Kafka Event
- Outbox Transaction Pattern 

---
메인 페이지 -> 마이 페이지 흐름
1. 사용자는 유효기간 별 포인트를 확인할 수 있다.
	1. 포인트는 지급일(시간 고려 X)로부터 5년 이내 사용해야한다.
	2. 토스 페이먼츠 결제를 통하여 포인트를 충전할 수 있다. (보너스 10% 제공, 최소 결제 금액 1,000원)

**고려사항**
- 토스 페이먼츠 결제 (외부 연동) 

-----
메인 페이지 -> 검색 흐름
1. 사용자는 상품을 검색할 수 있다.
	1. 특정 키워드가 들어가는 상품 중 5개만 노출한다.
	2. 이후 더보기 클릭 시 30개씩 나누어서 상품을 노출한다.
	3. 상품의 가격 정보는 사용자가 받을 수 있는 최대 할인 쿠폰이 적용된 가격을 노출한다.
		1. 상품 쿠폰
		2. 사용자 보유 쿠폰


**고려사항**
- 상품 이미지 최적화
- ElasticSearch 검색
- CDC
